Homotopic Skeletonization

Task:Perfom thinning on an image to find its skeleton

Steps:

  • Created eight structuring elements to remove boundaries of an image
  • To get the skeleton of an image, thinning operation has to be performed on the input image over multiple iterations
  • Hit or miss transform is performed using all the eight structuring elements in every iteration
  • The resultant thinned image after every iteration is compared with the resultant image in previous iteration, if both are same then the image cannot be thinned any more so the algorithm is halted.

Shape Analysis

Input: Image containing distinct objects and another image containing the same image but each object is rotated by certain angle.

Task: Perform shape analysis to find most similar objects.

Steps:

  • Bounding box of each object is computed
  • Calculated size distribution, pecstrum and shape complexity of each object
  • Distances between objects in each image are compared to select the most similar object.
