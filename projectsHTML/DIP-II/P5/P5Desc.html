Task: Generate fractal images using IFS

Steps:

  • Initialized an image of size 1024 x 1024
  • Initialized a starting point and ran IFS algorithm for a specified number of times
  • In each iteration, a random number is generated which is used to select the affine transform to be applied on the image
  • Set the pixel corresponding to this iteration to 1 and use it as seed point for the next iteration
